API Usage Guide

API Key: supersecretapikey123

সব API call এ এই key টি ?key=supersecretapikey123 হিসেবে query parameter এ দিতে হবে।

Important: যদি API key ভুল হয় বা না দেন, তাহলে 401 Unauthorized error পাবেন।

Error Code Meaning Solution
401 Invalid API key সঠিক API key ব্যবহার করুন
400 Bad Request Required parameters চেক করুন
404 Not Found Email বা message index চেক করুন
503 Redis not connected Redis server চালু করুন

JavaScript Example:
// Generate random email
fetch('http://165.22.97.51:3001/generate?key=supersecretapikey123')
  .then(response => response.json())
  .then(data => console.log(data.email));

// Get inbox
fetch('http://165.22.97.51:3001/inbox/test@oplex.online?key=supersecretapikey123')
  .then(response => response.json())
  .then(data => console.log(data.messages));
cURL Example:
curl "http://165.22.97.51:3001/generate?key=supersecretapikey123"
curl "http://165.22.97.51:3001/inbox/test@oplex.online?key=supersecretapikey123"
