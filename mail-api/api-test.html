📧 Email Management APIs

Generate, manage and view temporary emails

GET Generate Random Email
http://localhost:3001/generate?key=tempmail-master-key-2024&deviceId=test-device-123
Generates a random email address and stores it in MongoDB
GET Generate Custom Email
http://localhost:3001/generate/manual?key=tempmail-master-key-2024&username=myemail&domain=oplex.online&deviceId=test-device-123
Creates a custom email with specified username and domain
GET View Inbox
http://localhost:3001/inbox/myemail@oplex.online?key=tempmail-master-key-2024
View all messages for a specific email address
GET Check Email Availability
http://localhost:3001/check/testemail@oplex.online?key=tempmail-master-key-2024
Check if an email address is available for use
GET Email History
http://localhost:3001/history/test-device-123?key=tempmail-master-key-2024&page=1&limit=10
Get email history for a specific device with pagination
GET Starred Emails
http://localhost:3001/starred/test-device-123?key=tempmail-master-key-2024
Get all starred emails for a specific device
